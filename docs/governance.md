# Governance: Risks & Security

### 1. Risk Register

| Риск | Влияние | Детекция | Защита |
| :--- | :--- | :--- | :--- |
| Галлюцинация фактов | Высокое | Различие в ответах Debater A и B | Механизм дебатов и финальный Судья |
| Prompt Injection | Среднее | Узел Safeguard (SafeGuard Model) | Классификация входного запроса до запуска RAG |
| Ошибки парсинга таблиц | Среднее | Сравнение Markdown с исходным PDF | Использование Docling TableFormerMode.ACCURATE |
| Утечка ключей | Критическое | GitHub Secret Scanning | Использование .env и .gitignore |

### 2. Безопасность и данные
*   **Анонимизация:** Все запросы проходят через Safeguard-модель.
*   **Контроль:** Инструменты поиска (MCP) работают только в режиме чтения.
*   **Мониторинг:** Каждый шаг агента логируется в LangFuse для последующего аудита.

# SciVerify Agent: Autonomous Research Assistant

### 📕 О проекте
**SciVerify Agent** — это продвинутая агентная система на базе RAG (Retrieval-Augmented Generation), разработанная для глубокого анализа научных публикаций из arXiv. Система не просто ищет информацию, но и верифицирует её через состязательный процесс между несколькими агентами.

*   **Задача:** Автоматизация поиска и критического анализа научных данных.
*   **Для кого:** Исследователи, студенты и ML-инженеры.
*   **Боль:** Обычные LLM часто галлюцинируют в цифрах и методологиях при чтении PDF. SciVerify минимизирует этот риск.

### 📋 Технологический стек
*   **Оркестрация:** LangGraph
*   **База знаний:** Qdrant (Vector DB)
*   **Инструменты:** MCP (Model Context Protocol), ArXiv API, Docling
*   **Мониторинг:** LangFuse

### 🚀 Ключевые возможности PoC
1.  **Safeguard:** Автоматическая фильтрация вредных запросов и классификация намерений.
2.  **Multi-Index Retrieval:** Поиск по 5 типам индексов (чанки, саммари, вопросы, заголовки).
3.  **HyDE:** Улучшение поиска через генерацию гипотетических ответов.
4.  **Multi-Agent Debate:** Состязательная логика "Автор-Критик" для выявления ошибок в ответах.
5.  **Final Judge:** Вынесение итогового вердикта на основе проверенных фактов.

### 🛠 Установка (кратко)
